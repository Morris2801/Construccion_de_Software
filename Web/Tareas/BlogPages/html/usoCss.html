Elementos importantes
Parte del Ruleset Descripción
Selector: Define a qué elemento se le aplicará el estilo (admite comas: ul, h1, ...)
Declaración Aclara qué propiedad y qué valor se está aplicando.
Propiedad Especifíca qué atributo o característica de la declaración se está tocando por elemento.
Valor Escoge cómo se modificará el atributo seleccionado por la propiedad.
{} Todo ruleset debe estar entre llaves.
: Toda propiedad debe estar separada de su valor por ":"
; Toda declaración debe estar separada de la siguiente por ";"


Variedad y Jerarquía de selectores

En CSS, para ser más selectivo y específico en cuanto a qué se le está aplicando el estilo, dentro de los tags de un elemento particular se pueden puntualizar grupos o casos únicos a los cuales estilizar de determinada manera.
Esto es con selectores por Tipo, Clase, e Identificador.

  • Selector por Tipo

    • Aplica a todos los tags/elementos del mismo tipo: h1, p, a, main, div, nav, ... No se requiere gran modificación a la sintaxis usual.

  • Selector por Clase

    • Asignada a una clase, especificada dentro del tag como atributo h1 class = "clase1"...
    Para aplicar a todos los elementos de una clase, se usa un punto antes del nombre de la clase en el css: .clase1 {...}

  • Selector por Identificador

    • Asignada a un solo elemento, especificada dentro del tag como atributo h1 id = "id1"...
    Para aplicar a un solo elemento de una clase, se usa un "#" antes del nombre del id en el css: #id1
