Comandos frecuentes de Git y GitHub.

Comandos de Terminal

Comando Uso
pwd Devuelve ubicación actual de terminal en carpetas.
ls {-a} Muestra los archivos (con -a se muestran hasta los ocultos) dentro de la actual carpeta.
cd {path} Se cambia la carpeta foco en la que se concentra la terminal (con .. se sube a la carpeta padre).
mkdir {directory} Se crea una carpeta vacía.
touch {filename} Se crea un archivo del nombre especificado.
cat {filename} Imprime el archivo especificado completo, si es comprehensible.
head {filename} Muestra las primeras líneas de un archivo.
vi {filename} Abre un pequeño editor en terminal para hacer cambios rápidos a un archivo {i: insert, w: write, q: quit}.
rm {filename} Borra lo especificado.
mv {filename} {directory} Cambia ubicación de archivo. {si no se pasa un path, sino un nombre en el 2do argumento, se renombra el archivo del 1er argumento}.
