Comandos frecuentes de Git y GitHub.

Con esto pareces hacker.

Se requiere tener un repositorio en GitHub. Se puede crear con +Create Repository, y formateando lo necesario de detalles.


  • git log
  • git pull {origin} {branch}
  • git fetch -a
  • git clone {repository SSH key}
  • git add {filename}
  • git commit -m {filename}
  • git push {origin} {branch}
  • git checkout -b {branch}
  • git branch -d {branch}
  • git restore --staged {filename}
  • git restore {filename}
  • git merge {branch}
